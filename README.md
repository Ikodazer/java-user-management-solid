# Overview
This project is a simple user management system built in JavaFX to practice
software architecture principles such as SOLID and layered design.

The application simulates a small data persistence layer using an in-memory
database and separates responsibilities using a Service and Repository
architecture.

## Project Goals
- Practice SOLID principles
- Implement a layered architecture
- Separate business logic from data access
- Simulate a persistence layer using in-memory storage

## Features
- User CRUD system
- In-memory data storage
- Layered architecture using Service and Repository patterns
- Separation of concerns following SOLID principles
